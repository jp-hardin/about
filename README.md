## Custom Domain (Optional)

To connect a custom domain (e.g., `jaredhardin.com`):

1. Purchase the domain from any registrar
2. In this repo, go to **Settings → Pages → Custom domain** and enter the domain
3. At your registrar, add a CNAME record pointing to `[username].github.io`
4. Enable **Enforce HTTPS** in GitHub Pages settings
